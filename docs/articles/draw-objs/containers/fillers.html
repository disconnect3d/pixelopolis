Fillers¶

Renders many draw-objs in specified direction. Unline series, filler try to fill specified space with draw-objs.

Properties¶

display: v-filler | d-filler | h-filler;
Required
YES
Type v-filler

filler on X (width) axis

Type d-filler

filler on Y (depth) axis

Type h-filler

filler on Z (height) axis

v-series¶

d-series¶

h-series¶
body: SELECTOR-TO-DRAW-OBJ;

Links to draw-obj to fill the specified space.

padding: INTEGER;
Default value
0

Minimal padding between objects.

align: start | end | center | stretch;
Default value
start

align of objects in specified axis.

For this property is neccessary to set width/depth/height (depends on type)

align: start¶

align: end¶

align: center¶

align: stretch¶
(v-filler) justify: start | end | center | random;
(d-filler) justify: start | end | center | random;
(h-filler) justify: JUSTIFY-X, JUSTIFY-Y | center | random;
Default value
start

align of objects for the rest axis.

  • for v-filler: align of Y (depth) axis

  • for d-filler: align of X (width) axis

  • for h-filler: align of Y and X axis (with two arguments)

justify: start¶

justify: end¶

justify: center¶

justify: random¶

justify: start, center¶

justify: end, center¶

Example¶

row {
    display: v-filler;
    width: 300;
    depth: 100;
    align: stretch;

    body: house.red;
}
