Cylinder¶

Great for body’s tower or ancient area.

Properties¶

display: cone;
Required
YES
roof: SELECTOR-TO-WALL | null;
Default value
null

wall of top disk.

wall: SELECTOR-TO-WALL | null;
Default value
null

wall of side faces stretched vertically.

sides: INTEGER;
Default value
8

number of cylinder sides. If have more sides then is more circular:

sides: 6¶

sides: 32¶

Example¶

ancient-area {
    display: cylinder;
    width: 100;
    depth: 100;
    height: 100;

    roof: wall.roof;
    wall: wall.brick;
    sides: 32;
}

You can try in on https://pixelopolis.herokuapp.com/examples/Basic-shapes/Cylinder
