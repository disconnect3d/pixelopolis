Triangle¶

Just a triangle shaped roof.

Properties¶

display: triangle;
Required
YES
wall: SELECTOR-TO-WALL | null;
Default value
null

wall of side triangles.

roof: SELECTOR-TO-WALL | null;
Default value
null

wall of top faces.

Example¶

triangle-roof {
    display: triangle;
    width: 100;
    depth: 100;
    height: 100;

    roof: wall.roof;
    wall: wall.brick;
}

You can try in on https://pixelopolis.herokuapp.com/examples/Basic-shapes/Triangle
