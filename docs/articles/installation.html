Installation¶

Requirements¶

  • YACC

  • Bison

  • libpng

  • gcc

  • probably linux :)

Locally¶

Debug version:

make all

Prod version:

make all VERSION=PROD

Usage¶

./pixelopolis [CSS_PATH] [OUT_PNG_PATH]

Docker Version¶

# bulding image
docker build . -t pixelopolis

# running webpanel on localhost:8000
docker run -it --rm -p 8000:80 pixelopolis
