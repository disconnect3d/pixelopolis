World
=========

First element in file, describes width, height and select first draw-obj as root


Properties
------------

.. css:property:: width
    :type: INTEGER
    :required:

    width of image.

.. css:property:: height
    :type: INTEGER
    :required:

    height of image.

.. css:property:: body
    :type: DRAW-OBJ-SELECTOR
    :required:

    links the first draw obj. Usually is a container with another draw-objs.
